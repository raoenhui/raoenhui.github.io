# AI体检报告解读系统（纯前端版）

基于DeepSeek大模型的智能体检报告解读平台，为平安好医生提供专业的AI医疗健康服务。这是一个纯前端应用，直接在浏览器中调用DeepSeek API。

## 🏥 项目概述

本项目是一个专业的AI体检报告解读系统，能够：

- 📝 支持文本输入体检报告内容
- 🤖 通过DeepSeek大模型进行专业医学解读
- 📋 生成结构化的健康报告
- 💡 提供个性化的健康管理建议
- 🏥 推荐合适的健康服务产品

## ✨ 核心功能

### 1. 异常指标分析
- 精准识别异常体检指标
- 评估严重程度（轻度/中度/重度/危急）
- 多指标联动分析
- 健康趋势预测

### 2. 健康管理方案
- 合理的用药建议
- 个性化的生活方式指导
- 长期的健康管理计划
- 定期随访建议

### 3. 健康服务推荐
- 基于用户需求的智能推荐
- 平安好医生内部服务资源
- 2-5个最适合的健康服务

## 🛠️ 技术架构

### 纯前端技术
- **HTML5** + **CSS3** + **JavaScript**
- 响应式设计，支持移动端
- 现代化的UI界面设计
- 直接调用DeepSeek API

### 主要特性
- 无需后端服务器
- 单HTML文件包含所有功能
- 直接在浏览器中运行
- 支持文件上传和表单处理

## 🚀 快速开始

### 环境要求
只需要现代浏览器即可运行：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### 使用步骤

1. **打开应用**
   直接双击 `index.html` 文件或在浏览器中打开

2. **输入体检报告**
   - 在文本框中输入或粘贴体检报告内容
   - 可填写健康信息表单（可选）

3. **开始分析**
   - 点击"开始AI分析解读"按钮
   - 系统将调用DeepSeek API进行分析

4. **查看结果**
   - 查看异常指标分析
   - 获取健康管理建议
   - 查看推荐的健康服务

## ⚠️ 注意事项

1. **医学免责**
   - 本系统提供的建议仅供参考
   - 不能替代专业医生的诊断和治疗
   - 重大疾病请及时就医

2. **数据安全**
   - 所有处理在浏览器端完成
   - 数据不会上传到其他服务器
   - 建议对敏感信息进行脱敏处理

3. **API限制**
   - DeepSeek API有调用频率限制
   - 建议控制文本内容长度
   - 过长的内容可能需要较长时间处理

4. **浏览器兼容性**
   - 需要现代浏览器支持
   - 需要启用JavaScript
   - 需要网络连接调用API

## 🔮 使用限制

- 由于跨域限制，建议在支持CORS的环境中使用
- API调用依赖网络连接质量
- 建议输入内容控制在合理长度范围内
